Getting started

To read Johann's documentation, first build the documentation by running
> make html
and go to the main documentation page.

To build the Johann program and a default database of facts, first edit the first few lines of cpp/makefile to select your target system, and then run
> make johann
> make default
this will enter the Johann console and begin building the database default. After a while (1 minute -- 1 hour depending on your patience), type
(J) !save
and exit the Johann console by hitting CTRL+D or typing
(J) exit
You now have a default database (located as data/default.*)

Running Johann

To start the Johann console, run
> bin/johann
from the topmost directory. You can type
(J) help
for help on commands, or read the help page.

To update the database statistics page in the Johann console, type
(J) !stats

To view database maps, first build the jmapper map viewier with
> make jmapper
Then in the johann console, load your database by typing, e.g.,
(J) !load my_database
and save a map file (to stats/default.map) by typing
(J) !map
Finally you can run
> bin/jmapper

Writing jtext scripts

You can add .jtext scripts in scripts/user/. Then in Johann you can, e.g., import user/example.jtext by typing
(J) !import user/example.
See the scripts in scripts/* for examples of notation.

Reporting problems

If you find problems installing or running Johann (or better, want to suggest solutions to these problems), please email the author.
